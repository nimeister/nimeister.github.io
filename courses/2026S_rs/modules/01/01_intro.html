Lab#

  • Sign up for Google Earth Engine

  • Finish Intro to JavaScript for Earth Engine Tutorial Once you have your GEE account ready, type code.earthengine.google.com to start code editior, type the commands on the tutrial in code editor and try to run your code to see if you can get the same outputs.

  • Make sure you know how to print, create string, number and list and Objectives and functions in java script. Once you are done with the tutorial, create a function to calculate NDVI with inputs b4 and b3. Calculate NDVI with b4=0.6 and b3=0.2, output the NDVI values. Submit the link of your gee code to BB.

  • Watch this video - Since we didn’t have chance to play this video in class, you watch yourself, and write a short summary on what GEE is and what it can do and submit it to BB.
