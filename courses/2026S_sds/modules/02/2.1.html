Python Variables and Data Types#

One basic element for any programming language is variables. Variables can store data/objects of different types, which can do different things and have their own built-in functions.

Python has the following data/object types built in by default

  • 1. Numbers

  • 2. Boolean

  • 3. None

  • 4. Strings

  • 5. Sequence

  • 6. Mapping

  • 7. Binary
