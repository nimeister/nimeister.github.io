This is an Affine transformation matrix that defines how pixel coordinates map to real-world coordinates in your raster.

Value

Meaning

0.5

Pixel width (0.5 ft)

0.0

Row rotation (0 = no rotation)

990000.0

X-coordinate of upper-left corner

0.0

Column rotation (0 = no rotation)

-0.5

Pixel height (-0.5 ft, negative = north-up)

222500.0

Y-coordinate of upper-left corner

From .transform, you can get

  • Resolution: 0.5ft × 0.5ft pixels

  • Origin: Upper-left corner is at coordinates

  • Orientation: North-up, not rotated

  • CRS: Projected coordinate system
