Overview#

This course offers students the opportunity to learn the latest data science computing and visualization methods through processing and interpreting large-scale publicly available environmental datasets. Students will learn to use Python open-source tools to explore, visualize, and analyze large spatio-temporal environmental data available from various environmental data networks.

Highlights:

  • Explore fundamental concepts and real-world data science applications

  • Learn how to programatically answer real-world questions using remote sensing data

  • Machine learning

Weekly Schedule#

The course is organized into several week-long modules. Each module contains background reading assignments and Jupyter notebooks with an introduction, demo, and lab exercises.

  1. Data Science and Jupyter Notebook

  2. Basic Python 1

  3. Basic Python 2

  4. Pandas and numpy 1

  5. Pandas and numpy 2

  6. Data Visualization

  7. Midterm

  8. Geopandas

  9. Leaflet

  10. Rasterio

  11. Rioxarray

  12. From notebooks to packages

Structure#

Weekly workflow:#

  • We meet every Thursady. 2:30 pm - 5:15 pm. We follow the folling workflow:

    • Initial 15-30 minutes to discuss lab exercises

    • Next 30-60 minutes for introduction and demo for new material

    • Students finish exercises for homework (due the following week)

Midterm#

  • I propose a few research topics with code to access these data and possible research questions.

  • You write Python code to answer those questions.

  • Final deliverables: Jupyter notebook

Final Project#

  • Students propose, refine, perform and present independent or group projects

  • Final deliverables: Jupyter notebook and ~5-10 minute presentation
