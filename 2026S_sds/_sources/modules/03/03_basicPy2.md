### Starting a Jupyter Notebook Document

Click Start Menu on your computer, search for Miniforge Prompt to start a command-based window. You might want to navigate to the directory that you want to work on using the commands we learn [last week](../01/01_intro.md). Then you need to activate your sds environment first, then launch Jupyter Lab. 

Under the Mini Prompt window, type:

`conda activate sds`, hit `enter` key, then type

`Jupyter lab`, hit `enter` key

You can also double-click the text.bat file , which stores the commands like this in the directory you want to run JupyterLab:
```
@echo off
set MINIFORGE_PATH=%USERPROFILE%\miniforge3
call "%MINIFORGE_PATH%\Scripts\activate.bat" "%MINIFORGE_PATH%"
call conda activate sds
jupyter lab
pause
```
