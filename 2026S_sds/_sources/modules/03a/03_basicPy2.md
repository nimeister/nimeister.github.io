### Starting a Jupyter Notebook Document
Last week, we installed Anaconda or Minconda and created a sds working environment on your computer and learned how to launch Jupyter Lab directly. This week, we will use JupyterLab to write Jupyter notebooks. If you are using our lab computers for this class, try to work on the same computer you installed Jupyter Lab last week. Otherwise, you will have to reinstall it on a different computer. 

Click Start Menu on your computer, search for Anaconda/miniconda->Anaconda Prompt to start a command environment. You might want to navigate to the directory that you want to work on using the commands we learn [last week](../01/01_intro.md). Then you need to activate your sds environment first, then launch Jupyter Lab. 

Under the Anaconda Prompt window, type:

`conda activate sds`, hit `enter` key, then type

`Jupyter lab`, hit `enter` key
